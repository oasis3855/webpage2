著作権が消滅したと思われる「明治20年（1887年）内務省発行 大阪実測図」を写真撮影したものを掲載しています。

著作権の保護期間が経過していることの確認

図版の著作物の保護期間は1971年改正の著作権法では50年（団体の場合は発表時よりカウント、個人の場合は個人死亡時からカウント）。旧法を含めた著作権の保護期間はWikipediaの著作権の保護期間の説明によれば、次のようなものになると言われている。
