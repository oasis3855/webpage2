旅の持ち物リスト (個人的資料) Packing list

長期海外旅行の持ち物リスト Packing list for Travel (2017/06/15)
2週間以上の海外旅行に必要な旅の持ち物。食料は現地調達・自炊、消耗品は現地調達・処分、洗濯はコインランドリーが前提で、50リットルザックにすべて収まる範囲内でセレクトしている

数日の小屋泊・日帰りの軽登山に必要な持ち物リスト Packing list for Hiking (2024/08/02)
数日の小屋泊や日帰りの軽登山。長距離バスや鉄道を使って北アルプス等に出かけるときに、25リットルザックに収まる範囲内の荷物をセレクト。食料はすべて持参し、現地で飲食支出はしない前提
