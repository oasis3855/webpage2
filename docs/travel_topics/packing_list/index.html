長期海外旅行の持ち物リスト(個人的資料) Packing list for Travel

Home > 旅日記と世界の写真 > 旅の情報 > このページ

Last Update 2017/06/15

 

 

2週間以上の海外旅行に必要な旅の持ち物。食料はすべて現地調達、所持物が消耗した場合は現地の安売りスーパー等で買い替え、衣服はコインランドリーで洗濯。 この持ち物で1ヶ月弱の旅行をした個人的経験から、ほぼ∞（無限大）日数の旅行にも適用可能。

日本円（最低限 通常２万円程度）
USドルまたはユーロ（日本円が両替不可能な場所）

VISA および MASTER 双方
